# Url-Manipulator

## Requirment:
### packages 
- With_Statement
- contextlib
- urllib.request
- sys
- bcolors
- argparse
- requests

### python > 3.x 


## usage: 

url-manipulator.py [-h] --help

OPTIONS:
```
-h,      --help    
             	 show this help message and exit
-s       Path of File with Long URLS
		       	<To Short URLS>
-f 	     Path of File with short URLS 
	      		<To Get Long URLS>
```
